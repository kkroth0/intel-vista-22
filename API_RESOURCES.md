# Threat Intelligence API Resources

This document lists the APIs used in the Threat Intelligence Dashboard, along with links to their documentation and where to obtain API keys.

## 1. VirusTotal
*   **Website:** [https://www.virustotal.com/](https://www.virustotal.com/)
*   **API Documentation:** [https://developers.virustotal.com/reference/overview](https://developers.virustotal.com/reference/overview)
*   **Get API Key:** Sign up for a free account to get a public API key.

## 2. AbuseIPDB
*   **Website:** [https://www.abuseipdb.com/](https://www.abuseipdb.com/)
*   **API Documentation:** [https://docs.abuseipdb.com/](https://docs.abuseipdb.com/)
*   **Get API Key:** Create an account and generate a key in your profile.

## 3. Fortinet FortiGuard
*   **Website:** [https://www.fortiguard.com/](https://www.fortiguard.com/)
*   **API Documentation:** [https://fndn.fortinet.net/](https://fndn.fortinet.net/) (Requires Fortinet Developer Network access)
*   **Note:** Public API access is limited; often requires enterprise subscription.

## 4. Kaspersky Threat Intelligence
*   **Website:** [https://www.kaspersky.com/enterprise-security/threat-intelligence](https://www.kaspersky.com/enterprise-security/threat-intelligence)
*   **API Documentation:** [https://opentip.kaspersky.com/](https://opentip.kaspersky.com/) (OpenTIP)
*   **Get API Key:** Check OpenTIP for free limits or contact sales for enterprise access.

## 5. Shodan
*   **Website:** [https://shodan.io/](https://shodan.io/)
*   **API Documentation:** [https://developer.shodan.io/](https://developer.shodan.io/)
*   **Get API Key:** Create an account. Free accounts have limited credits.

## 6. URLhaus (Abuse.ch)
*   **Website:** [https://urlhaus.abuse.ch/](https://urlhaus.abuse.ch/)
*   **API Documentation:** [https://urlhaus-api.abuse.ch/](https://urlhaus-api.abuse.ch/)
*   **Get API Key:** Open access for some endpoints; others may require request.

## 7. ThreatFox (Abuse.ch)
*   **Website:** [https://threatfox.abuse.ch/](https://threatfox.abuse.ch/)
*   **API Documentation:** [https://threatfox.abuse.ch/api/](https://threatfox.abuse.ch/api/)
*   **Get API Key:** Login with Twitter/GitHub to get an API key for submissions, but reading is often open.

## 8. Hybrid Analysis
*   **Website:** [https://www.hybrid-analysis.com/](https://www.hybrid-analysis.com/)
*   **API Documentation:** [https://www.hybrid-analysis.com/docs/api/v2](https://www.hybrid-analysis.com/docs/api/v2)
*   **Get API Key:** Sign up for a free account.

## 9. AlienVault OTX
*   **Website:** [https://otx.alienvault.com/](https://otx.alienvault.com/)
*   **API Documentation:** [https://otx.alienvault.com/api](https://otx.alienvault.com/api)
*   **Get API Key:** Sign up for a free account.

## 10. Cisco Talos
*   **Website:** [https://talosintelligence.com/](https://talosintelligence.com/)
*   **API Documentation:** Limited public API. Often requires Cisco products or specific agreements.
*   **Note:** Web reputation lookups are available via their site.

## 11. IBM X-Force Exchange
*   **Website:** [https://exchange.xforce.ibmcloud.com/](https://exchange.xforce.ibmcloud.com/)
*   **API Documentation:** [https://api.xforce.ibmcloud.com/doc/](https://api.xforce.ibmcloud.com/doc/)
*   **Get API Key:** Sign up for an IBM ID and generate a key.



## 14. Spamhaus
*   **Website:** [https://www.spamhaus.org/](https://www.spamhaus.org/)
*   **API Documentation:** [https://docs.spamhaus.com/](https://docs.spamhaus.com/)
*   **Get API Key:** Requires subscription for API access (DQS).

## 15. MalwareBazaar (Abuse.ch)
*   **Website:** [https://bazaar.abuse.ch/](https://bazaar.abuse.ch/)
*   **API Documentation:** [https://bazaar.abuse.ch/api/](https://bazaar.abuse.ch/api/)
*   **Get API Key:** Open for queries.



## 17. PhishTank

*   **API Documentation:** [https://www.phishtank.com/developer_info.php](https://www.phishtank.com/developer_info.php)
*   **Get API Key:** Register for an application key.

## 18. CIRCL PassiveDNS
*   **Website:** [https://www.circl.lu/services/passive-dns/](https://www.circl.lu/services/passive-dns/)
*   **API Documentation:** [https://www.circl.lu/doc/misp/automation/](https://www.circl.lu/doc/misp/automation/) (and other CIRCL docs)

## 19. Pulsedive

*   **API Documentation:** [https://pulsedive.com/api/](https://pulsedive.com/api/)

## 20. ThreatCrowd

*   **API Documentation:** [https://search.censys.io/api](https://search.censys.io/api)
